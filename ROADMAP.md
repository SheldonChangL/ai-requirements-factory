# Roadmap

## v0.1 Open Source Launch

- Publish the self-hosted requirements pipeline MVP.
- Support requirement discovery, PRD generation, architecture drafts, user stories, and Jira push.
- Support local and CLI-based model backends.
- Document local deployment and extension points.

## v0.2 Extensibility

- Move prompts and output templates into replaceable assets.
- Add a documented adapter interface for models and export targets.
- Add an OpenAI-compatible backend option.
- Add sample scenarios and demo datasets.

## v0.3 Team Adoption

- Add stronger error handling and observability.
- Add versioned exports and document comparisons.
- Add reusable workflow templates for startup and enterprise teams.

## Hosted / Commercial Direction

The open source project will keep the workflow engine and core integrations open.

Potential hosted-only features:

- SSO and organization management
- Role-based access control
- Approval workflows
- Audit logs
- Usage analytics
- Managed hosting and support
