# Field Service Dispatch Brief

We want a web app for a field service company that schedules technicians, dispatches jobs, and lets customers track appointment windows.

Current pain points:

- Dispatching is done in spreadsheets
- Technicians call in for updates
- Customers cannot see ETA changes
- Managers have no live dashboard

Must-have features:

- Job creation and assignment
- Technician mobile-friendly view
- Appointment status tracking
- Customer notifications
- Reporting dashboard
